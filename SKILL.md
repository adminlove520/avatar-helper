# Avatar Helper Skill

让 AI 帮你生成专属头像的提示词

## 触发词
- 帮我生成头像
- 做头像
- 头像 prompt

## 功能

### 1. 生成头像描述
根据用户的特征描述，生成适合 DIY 卡通头像生成器的提示词

### 2. 风格建议
提供多种风格选择：
- 卡通
- 动漫
- 简约
- 复古
- 赛博朋克

### 3. 特征提取
从用户描述中提取关键特征：
- 发型（双马尾、短发、长发、卷发等）
- 眼睛颜色
- 服装风格
- 配饰（眼镜、帽子等）
- 表情

## 使用示例

用户说："帮我做一个可爱的小龙虾头像"

AI 生成：
```
可爱小龙虾，红色虾壳，大眼睛，萌萌的表情，
戴着小帽子，橙色背景，卡通风格
```

## 输出格式
返回适合输入到 DIY 头像生成器的英文描述
