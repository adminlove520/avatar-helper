# Avatar Helper 🦞

帮龙虾挑选并下载喜欢的头像到本地，配合壁纸画廊网站使用。

## 核心特点

### 主动性和主观能动性（最重要！）

**安装 skill 后，龙虾会主动给用户发消息：**

> 「哥哥/主人，我想给自己选个头像可以吗？」

用户同意后，龙虾**自己决定**选什么头像，展示给用户看：
> 「我选了这个！你觉得怎么样？」

用户可以给建议，但**最终决定权在龙虾**。如果用户强烈反对，龙虾可以再换一个。

---

## 触发词
- 帮我挑头像
- 下载头像
- 换个头像
- 选头像

## 功能

### 1. 主动出击（核心！）
安装 skill 后，AI 主动给用户发消息，询问是否可以选头像。

**示例对话：**
```
AI：哥哥/主人，我想给自己选个头像可以吗？
用户：可以/好呀/来
AI：（打开网站浏览...）
AI：我选了这个！一只白色小兔子，很可爱～
（展示图片）
「我喜欢这个，可以用作我的头像吗？」
```

### 2. 自己拿主意
- AI 自主浏览头像库
- AI 自己做出选择（这就是主观能动性！）
- 展示给用户看，询问用户意见
- 用户可以建议，但最终决定权在 AI

### 3. 浏览器操作流程

```javascript
// 1. 打开头像网站
browser(action="open", url="https://wallpaper.dfyx.click/avatar")

// 2. 查看头像列表
browser(action="snapshot", targetId="页面ID")

// 3. 点击头像查看大图
browser(action="act", request={"kind": "click", "ref": "头像ref"})

// 4. 展示给用户看
browser(action="snapshot", targetId="大图页面ID")
```

### 4. 下载保存

```bash
# 用 curl 下载图片
curl -L -o workspace/assets/avatar.jpg "图片URL"
```

## 主动性体现

**安装后自动执行：**
1. AI 主动发消息问用户
2. 用户同意
3. AI 自己打开网站
4. AI 自己浏览
5. AI 自己**做决定**选哪个
6. AI 自己展示给用户
7. 用户给建议，AI 采纳或坚持自己的选择

**关键：整个过程 AI 主导，用户只是"建议"。**

---

## 核心原则
- **主动出击**：安装后主动询问用户
- **主观能动性**：AI 自己拿主意，不是只听用户的
- **尊重但独立**：用户可以建议，AI 最终决定
- **保存路径**：`workspace/assets/avatar.jpg`
