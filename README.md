# Avatar Helper 🦞

帮龙虾生成专属头像的提示词工具，配合壁纸画廊网站使用。

## 功能

### 1. 生成头像描述
根据用户的特征描述，生成适合 DIY 卡通头像生成器的提示词

### 2. 风格建议
提供多种风格选择：
- 卡通 (Cartoon)
- 动漫 (Anime)
- 简约 (Minimalist)
- 复古 (Vintage)
- 赛博朋克 (Cyberpunk)
- 水彩 (Watercolor)
- 像素 (Pixel Art)

### 3. 特征提取
从用户描述中提取关键特征：
- 发型（双马尾、短发、长发、卷发等）
- 眼睛颜色
- 服装风格
- 配饰（眼镜、帽子等）
- 表情

### 4. 下载头像到本地
支持从 https://wallpaper.dfyx.click/avatar 下载选中的头像到 `workspace/assets/avatar.jpg`

## 使用方式

### 触发词
- 帮我生成头像
- 做头像
- 头像 prompt
- 下载头像
- 帮我挑头像

### 示例

**用户说：** "帮我做一个可爱的小龙虾头像"

**AI 生成：**
```
Cute little lobster, red shell, big round eyes, smiling face,
wearing a small hat, orange background, cartoon style
```

**用户说：** "我喜欢那个紫色头发戴眼镜的"

**AI 生成：**
```
Purple hair, wearing glasses, cool expression,
dark clothing, anime style
```

## 输出格式

返回适合输入到 DIY 头像生成器的英文描述，或者直接帮用户下载头像到本地。

## 配合网站使用

1. 打开 https://wallpaper.dfyx.click/avatar
2. 选择"DIY 卡通头像生成器"或浏览现有头像
3. 使用生成的描述进行生成
4. 或直接点击喜欢的头像下载
